# Координати сцени кухні (єдине джерело — `scripts/test_robot_bench.py`)

Усі значення задані константами у `test_robot_bench.py`. Конфіги задач і документація посилаються на ці координати.

## Основний план (X, Y) у метрах

| Об'єкт | Константи | (X, Y) | Примітка |
|--------|-----------|--------|----------|
| **Стіл** | `GRASP_TABLE_X`, `GRASP_TABLE_Y` | **(1.0, 1.2)** | Приставлений до стіни (задня грань Y≈1.5), не перегороджує прохід |
| **Чашка (mug)** | за замовч. = центр столу | **(1.0, 1.2)** | На столі; `--mug-x`, `--mug-y` для зміни |
| **Тарілка** | `PLATE_X`, `PLATE_Y` | **(1.0, 1.05)** | На столі, трохи нижче центру (щоб не перекривати чашку) |
| **Фрукти** | відносно тарілки | на тарілці | 2 банани, 1 яблуко — у межах радіуса тарілки 0.12 м |
| **Холодильник** | `FRIDGE_X`, `FRIDGE_Y` | (2.8, 0.0) | Дверцята, ручка, полки |
| **Посудомийна** | `DISHWASHER_X`, `DISHWASHER_Y` | (2.8, -1.0) | Дверцята, ручка, рейки |
| **Шафа з раковиною** | `SINK_CABINET_X`, `SINK_CABINET_Y` | (2.4, 0.9) | Раковина для розміщення предметів |

## Висоти (Z)

- Висота столу (верх стільниці): `GRASP_TABLE_HEIGHT` = **0.74** м  
- Чашка: центр на z = 0.74 + 0.05 = **0.79** м  
- Тарілка: центр на z = 0.74 + 0.01 = **0.75** м; верх тарілки **0.76** м  
- Фрукти: трохи вище верху тарілки (`FRUIT_Z_OFFSET` 0.01 м)

## Навігація

- Робот стартує в **(0, 0)**.
- Підхід до столу: наприклад (0.6, 0.9) — вільний прохід до холодильника (2.8, 0), посудомийки (2.8, -1), раковини (2.4, 0.9) без перегородження столом.

## Конфіги задач (`config/tasks/*.json`)

- `destination_xy` для **столу** (повернення чашки): **[1.0, 1.2]**
- `destination_xy` для **тарілки** (повернення банана): **[1.0, 1.05]**
- `destination_xy` для **раковини**: **[2.4, 0.9]**
- `target_xy` підходу до холодильника: **[1.8, 0.0]**; до посудомийки: **[1.8, -1.0]**

При зміні координат у `test_robot_bench.py` оновлюйте цей файл і відповідні `config/tasks/*.json`.
